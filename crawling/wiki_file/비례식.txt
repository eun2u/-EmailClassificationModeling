비례식(比例式, 영어: proportional expression)은 
  
    
      
        a
        ,
        b
      
    
    {\displaystyle a,b}
  의  비와 
  
    
      
        c
        ,
        d
      
    
    {\displaystyle c,d}
  의 비가 같은 경우, 두 비를 등호(=)를 이용하여 나타낸 수식을 말한다.
즉,비례식은 두  비

  
    
      
        a
        :
        b
      
    
    {\displaystyle a:b}
  와
  
    
      
        c
        :
        d
      
    
    {\displaystyle c:d}
  가 같은 경우를 등호를 이용하여 수식으로 나타내는것을 목적으로 하는 식인데,
수학에서 비례식을 
  
    
      
        a
        :
        b
        =
        c
        :
        d
      
    
    {\displaystyle a:b=c:d}
  의 꼴로 표현하기로 약속했다.

비례식의 내항과 외항
비례식의 기초가 되는  비에서 항을 분류하는 방법은
a와 b의 비 
  
    
      
        a
        :
        b
      
    
    {\displaystyle a:b}
  에서 기호 : 를 기준으로 : 의 앞에있는 항 
  
    
      
        a
      
    
    {\displaystyle a}
  를 전항, 뒤에 있는 항 
  
    
      
        b
      
    
    {\displaystyle b}
  를 후항이라고 부른다. 비례식에서도 '어떤 기호'를 기준으로 각 위치에 따라 항을 구분하여 부른다.
비례식 
  
    
      
        a
        :
        b
        =
        c
        :
        d
      
    
    {\displaystyle a:b=c:d}
  에서 기호 = 을 기준으로 : 기호의 안쪽에 있는 두 항 b,c를 내항이라 부르고, : 기호의 바깥에 있는 두 항 a,d를 외항이라 부른다.

비례식의 성질
분수와 비례식의 관계
분수의 성질은

  
    
      
        
          
            a
            b
          
        
        =
        
          
            
              2

              2
              b

    {\displaystyle {\frac {a}{b}}={\frac {2a}{2b}}={\frac {3a}{3b}}}
  (a,b는 0이 아닌 실수)와 같이 분자와 분모의 비가 같으면, 그 분수의 값은 모두 같다.
이 성질은 비례식 
  
    
      
        a

    {\displaystyle a:b=2a:2b=3a:3b=4a:4b}
  과 매우 밀접한관계에 있다.
위의 성질로부터, 분수는 분자,분모가 0 이 아닐때, 비로 나타낼 수 있음을 뜻한다.
하나 주의할점은 수학에서는 a:b의 항 a,b는 0이 아닌 정수여야 비로 인정해준다는것이다. 만약 항 a,b가 유리수라면, 비의 전항과 후항에 적당한 수를 곱하여 정수로 바꿀 수 있다는 성질이 있기때문이며, a와b가 무리수 혹은 허수이면 그 비를 어떤 값으로 나타낼 수 없어 수학적가치가 떨어진다. 따라서 a,b는 무조건 0이 아닌 정수로 나타내야한다.
분수와 비례식의 관계로부터 알 수 있는 성질로 비례식의 내항의 곱과 외항의 곱은 같다.
비례식 
  
    
      
        a
        :
        b
        =
        c
        :
        d
      
    
    {\displaystyle a:b=c:d}
  는 
  
    
      
        a
        :
        b
        =
        m
        a
        :

        c

    {\displaystyle a:b=ma:mb=c:d}
  로 볼 수 있다. (m은 비례상수),(a,b,c,d는 0이 아닌 실수)
즉, 위의 비례식은 분수로 나타낼 수 있음을 뜻한다.
위의 비례식을 분수들로 나타내면

  
    
      
        
          
            a
            b

            c

   (어떤 항을 분자의 자리에 보내든 상관은 없다.)이므로 
  
    
      
        a
        d
        =
        b
        c
      
    
    {\displaystyle ad=bc}
   인 등식이 만들어진다.
이 등식이 의미하는 성질은 그 어떤 비례식도 내항의 곱과 외항의 곱은 항상 같다는 것이다.
